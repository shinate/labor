SYSTEM REQUIREMENTS

OS: Windows XP or newer
CPU: 1.0 Ghz
RAM: 1 GB
Graphics: 256 RAM, with full support of openGL 2.0
Free disc space: 100 MB
Sound: openAL installed
.NET Framework 3.5
Microsoft Visual C++ 2010 Redistributable


INSTALATION

1) First install original Diablo 1 game (any patch) and then download the mod from the official website by clicking HERE.
2) Now copy all the files from the archive to your Diablo 1 game folder. (Note! openAL library is included in the archive)
3) Remember you have to insert Diablo 1 cd in drive or you can copy diabdat.mpq from cd to your Diablo 1 game folder.
4) And finally to start a Diablo 1 HD game just run the Belzebub.exe file.
